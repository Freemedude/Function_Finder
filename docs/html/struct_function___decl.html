Detailed Description

Contains information parsed on a source code function declaration.

Details

This struct is the core of the tool, it is what is parsed during importing, serialized when exported, and then again imported via the consumer-facing initialization function. The information stored is meant to enable the consumer to implement great tooling for whatever they need these commands for.
